# 🏆 Dota Stats by Ec9ins

Современное десктопное приложение для просмотра статистики игроков в Dota 2. Построено на Kotlin Multiplatform (Compose Desktop) с использованием принципов Clean Architecture и MVI.

![Compose](https://img.shields.io/badge/Compose_Desktop-1.7.3-green.svg)
![Ktor](https://img.shields.io/badge/Ktor-3.0.1-purple.svg?logo=ktor)

## ✨ Ключевые возможности

* **📊 Подробная статистика:** Просмотр ранга (с автоматической конвертацией в понятный формат: "Immortal", "Ancient 4" и т.д.), винрейта и региона.
* **🦸‍♂️ Сигнатурные герои:** Расчет топ-3 лучших героев игрока с отображением количества матчей и точного % побед.
* **🏅 Распознавание PRO-игроков:** Приложение автоматически определяет киберспортсменов, показывая их официальный никнейм и тег команды.
* **🎨 Смена тем (Material 3):** Поддержка динамической цветовой палитры. Плавное переключение между темами (Emerald, Ruby, Blue, Gold) в один клик.
* **🎬 Анимации:** Интерфейс обогащен компонентом `AnimatedContent`, обеспечивая приятный пользовательский опыт и плавные переходы между состояниями.

## 🏗 Архитектура проекта

Проект строго следует разделению на слои для обеспечения масштабируемости и тестируемости:
* `data/api/` — Сетевой слой. Работа с OpenDota API через **Ktor Client**.
* `data/model/` — Дата-классы. Безопасный парсинг JSON с помощью **Kotlinx Serialization**.
* `ui/` — Слой представления. Содержит Compose-экраны и `PlayerViewModel`, которая управляет состоянием через `StateFlow` и `sealed class` (MVI-подход).
