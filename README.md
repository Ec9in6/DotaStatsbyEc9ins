# Dota 2 Stats CLI 🎮

Консольное приложение на Kotlin для быстрого просмотра статистики игрока через OpenDota API.

## 🚀 Основные функции
* **Профиль:** Вывод никнейма и текущего ранга.
* **Статистика:** Общее количество побед и поражений.
* **Аналитика:** Автоматический расчет Winrate.
* **Интерфейс:** Цветной вывод (ANSI) для удобства чтения в терминале.

## 🛠 Технологии
* **Language:** Kotlin 2.0
* **Network:** Ktor Client (CIO)
* **JSON:** kotlinx.serialization
* **Build System:** Gradle

## 📖 Как запустить
1. Склонируйте репозиторий.
2. Откройте в IntelliJ IDEA.
3. Запустите `Main.kt`.
4. Введите Account ID (или нажмите Enter для загрузки профиля Miracle-).
